Allypatch is a softbotic (flexible robotic) intelligent, early-warning wearable patch that prevents heat-related complications and deaths. This device monitors multiple critical biomarkers and environmental data to determine heat stress and displays color-coded alerts via LED with haptic vibration feedback. The patch includes a PPG sensor for heart rate, temperature sensor for skin temperature, a 3-axis accelerometer to measure activity levels, and an ambient temperature and humidity sensor to collect environment data. The microcontroller utilizes all the sensor data to calculate core body temperature, sweat rate, and multivalidates early warnings to ensure no false alarms. Power is maintained through a battery and USB charging with a backup solar cell.
